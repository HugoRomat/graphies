Implementation

Graphies is a Web-based application, implemented with D3.js and paper.js.
The former handles all UI widgets in SVG, as well as many computations (visual mappings, force-directed layout, etc.), while the latter is used to draw the node-link diagram. We draw the diagram using 2D primitives directly in an HTML Canvas, as opposed to manipulating an SVG DOM hierarchy, for the sake of performance. The two UI components are seamlessly integrated using CSS rules and input event redirections
